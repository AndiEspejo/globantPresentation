# Presentación Globant

Aplicación de React con Vite para presentar información sobre Globant - análisis de sistemas de información.

## Características

- 9 slides interactivos con información sobre Globant
- Navegación con botones y indicadores de slides
- Diseño moderno con Tailwind CSS
- Animaciones y transiciones suaves
- Responsive design

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Build para Producción

```bash
npm run build
```

## Vista Previa de la Build

```bash
npm run preview
```

## Tecnologías Utilizadas

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (iconos)

## Contenido de la Presentación

1. Portada
2. ¿Qué es Globant?
3. Nuestros Clientes
4. Competidores Principales
5. Presencia en Redes Sociales
6. ¿Qué genera recordación?
7. Sistemas de Información (Parte 1)
8. Sistemas de Información (Parte 2)
9. Cierre

## Navegación

- Use los botones de flecha para navegar entre slides
- Click en los indicadores circulares para ir a un slide específico
- Las flechas se deshabilitan en el primer y último slide
